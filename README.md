# Pixel-crew


## Dependencies

Install [cargo-make](https://github.com/sagiegurari/cargo-make)
```sh
cargo install --force cargo-make
```

Yew
```sh
rustup target add wasm32-unknown-unknown
```
```sh
cargo install trunk
```

Npm  
Yarn 

Solana
```sh
sh -c "$(curl -sSfL https://release.solana.com/v1.10.32/install)"
```
